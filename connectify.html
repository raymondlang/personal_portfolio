Connectify

Full stack social media application that allows users to share posts and interact with friends.

  • GitHub
  • GitHub

Description

Connectify, a platform inspired by the likes of Facebook, offers users a seamless experience for sharing posts and connecting with friends.Connectify boasts a robust authentication system that's both comprehensive and secure. Making use of JWT (JSON Web Tokens) and a customized logic, this authentication component ensures user access.

Technologies

Approach

  • Implemented front-end and back-end, ensuring cohesive and efficient end-to-end functionality
  • Designed JWT and custom authentication logic highlights my competency in building secure user access systems, essential for safeguarding user data
  • Introducted globally customizable themes and dark mode to enhance user experience and create visually appeal that caters to diverse user preferences
  • Implemented React Dropzone to enable seamless and user-friendly file uploads, enhancing user interaction
